## Baseline Scope (v0.8.0)

The v0.8.0 baseline records the governance foundation used as bridge toward the
next release line:

- strict-vs-score governance boundaries formalized for CI usage;
- sovereign audit usage documented as maintainers' inspection path;
- namespace stability contract documented for integration tooling;
- deterministic runtime constraints and security posture documentation hardened.

## Historical Note

Post-freeze work discovered during the epoch shift is intentionally excluded from
this historical page and tracked in the v0.9.0 manifesto.

## Compatibility Snapshot

v0.8.0 remains a stable historical reference for contributors auditing legacy
repository states and migration decisions.
