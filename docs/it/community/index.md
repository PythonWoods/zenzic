# Community

## Partecipa

Come utente di Zenzic, sei in buona compagnia. Che tu stia validando la documentazione di un progetto software, gestendo una knowledge base o producendo altri contenuti per siti statici, molti altri condividono i tuoi interessi e sono pronti ad aiutarti a sfruttare al meglio Zenzic.

Il nostro team in crescita presso PythonWoods sta costruendo software Open Source robusto e adatto a un'ampia gamma di casi d'uso. Come ogni progetto Open Source, ci affidiamo ai contributi della nostra community per comprendere meglio le esigenze non soddisfatte e identificare eventuali problemi riscontrati dagli utenti.

## Come contribuire

In quanto software Open Source, Zenzic beneficia di diversi tipi di contributi dalla community dei suoi utenti. Il nostro approccio aperto ti permette di:

<div class="grid cards" markdown>

- :lucide-bug: &nbsp; __Segnalare bug__

    ---

    Segnala problemi tracciati nel nostro [issue tracker]. Il nostro team in crescita ci permette di fare triage e risolvere i problemi in modo tempestivo.

    [:lucide-arrow-right: Segnala un bug](contribute/report-a-bug.md)

- :material-file-edit-outline: &nbsp; __Migliorare la documentazione__

    ---

    Aiutaci a garantire che la documentazione sia coerente, comprensibile e che i lettori trovino rapidamente le informazioni che cercano.

    [:lucide-arrow-right: Segnala un problema nei docs](contribute/report-a-docs-issue.md)

- :material-lightbulb-on-outline: &nbsp; __Richiedere funzionalità__

    ---

    Crea richieste di modifica per garantire che Zenzic copra i casi d'uso comuni e offra il massimo valore agli utenti.

    [:lucide-arrow-right: Richiedi una modifica](contribute/request-a-change.md)

- :lucide-git-pull-request: &nbsp; __Contribuire al codice__

    ---

    Contribuisci allo sviluppo rispondendo a un'issue con una pull request che corregge un bug, introduce una nuova funzionalità o migliora le prestazioni.

    [:lucide-arrow-right: Apri una pull request](contribute/pull-requests.md)

</div>

## Citare Zenzic

Se utilizzi Zenzic nella tua ricerca, nelle pipeline di documentazione o in report tecnici e
desideri citarlo formalmente, il file [`CITATION.cff`][citation-cff] è presente nella root del
repository. La maggior parte degli strumenti di ricerca (GitHub, Zenodo, Zotero) può leggere
questo formato direttamente.

__GitHub__: clicca il pulsante __"Cite this repository"__ sulla [pagina del repository][repo] —
GitHub trasforma automaticamente il `CITATION.cff` in formato APA e BibTeX.

__BibTeX manuale:__

```text
@software{zenzic,
  author  = {PythonWoods},
  title   = {{Zenzic: The Agnostic Documentation Integrity Framework}},
  version = {0.5.0a4},
  date    = {2026-04-03},
  url     = {https://zenzic.pythonwoods.dev/},
  license = {Apache-2.0},
}
```

  [citation-cff]: https://github.com/PythonWoods/zenzic/blob/main/CITATION.cff
  [issue tracker]: https://github.com/PythonWoods/zenzic/issues
  [repo]:          https://github.com/PythonWoods/zenzic
