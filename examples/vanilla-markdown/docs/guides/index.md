# Guides

This section has an `index.md` — Zenzic considers it properly indexed and
will not report a `MISSING_DIRECTORY_INDEX` finding for this directory.

The directory index check fires only for subdirectories that contain Markdown
files but have no `index.md` landing page. Having an index guarantees that
visitors will always find a meaningful entry point when they navigate here.

See [Setup](setup.md) to get started.
