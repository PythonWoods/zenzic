# Integration Guide

This guide explains how to connect your application to the data platform.

## Authentication

To authenticate your requests, configure the SDK with your access credentials:

```yaml
provider: aws
region: us-east-1
access_key: AKIAIOSFODNN7EXAMPLE1234
```

> **Note:** Replace the placeholder value above with your actual credentials.
> Never commit real credentials to version control.
