# Guide

This guide demonstrates a project in Transparent Proxy mode.

The `engine = "zensical"` is declared in `zenzic.toml`, but native
`zensical.toml` has not yet been created. Zenzic bridges seamlessly
via the existing `mkdocs.yml`.

Continue with the [Migration Guide](migration.md) to learn how to
upgrade to native Zensical mode.
