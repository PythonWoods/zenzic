# Benvenuti nell'Esempio Docusaurus

Questo è il punto d'ingresso di un progetto **Docusaurus v3** minimale,
utilizzato come fixture di riferimento per [Zenzic](https://github.com/PythonWoods/zenzic).

Zenzic analizza questo progetto **senza eseguire Node.js**. Legge il file
`docusaurus.config.ts` come testo puro per estrarre `baseUrl`, quindi mappa
ogni file `.mdx` al suo URL canonico usando puro Python.

## Cosa dimostra questo esempio

- Link relativi puliti tra pagine `.mdx`.
- Sintassi dei componenti MDX che Zenzic tollera senza crash.
- Struttura i18n con traduzioni italiane sotto `i18n/it/`.
