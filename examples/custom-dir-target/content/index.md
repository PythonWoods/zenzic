# Welcome

This is the home page for a project whose Markdown source lives in `content/`
rather than the conventional `docs/` directory.

## About

Many static-site generators — Hugo, Docusaurus, Astro, Hexo — use `content/`
or `pages/` as the source root. Zenzic works with any of them: pass the
directory as a positional argument and the `StandaloneAdapter`
is selected automatically.

## Getting Started

See the [setup guide](guides/setup.md) for installation instructions,
configuration options, and your first build.
