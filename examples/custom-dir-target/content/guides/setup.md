# Setup Guide

This page walks through the initial setup and basic configuration.

## Requirements

- Python 3.10 or later
- A terminal with at least 80 columns for best display

## Installation

Install from PyPI using pip or uv:

```bash
pip install acme-project
# or
uv add acme-project
```

After installation, verify the version:

```bash
acme --version
```

## First Run

Navigate to your project directory and run the default command:

```bash
acme init
```

This creates a local configuration file and prints a short summary
of detected settings. No network access is required.
