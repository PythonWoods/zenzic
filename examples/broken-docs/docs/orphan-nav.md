# Nav Orphan (routing trap)

This page exists on disk but is intentionally **absent from the nav** in `mkdocs.yml`.

Any link from a public page to this file triggers **UNREACHABLE_LINK** — the file is
physically present and MkDocs will copy it to `site/`, but no user can discover it
through navigation. It is a "dark page".
