# Configuration

This document explains all available configuration options for the project.

## Core Settings

Set the `engine` field to match your documentation framework: `mkdocs`, `zensical`,
`docusaurus`, or `standalone`. The `docs_dir` field specifies the root directory
containing your Markdown source files.

## Advanced Options

For extended configuration, see the [API reference](../reference/api.md) which
documents all programmatic interfaces and their corresponding configuration keys.
Refer to the project's main documentation for engine-specific configuration examples
and recommended practices.
