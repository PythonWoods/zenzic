# Getting Started

This guide walks you through the initial setup of your project. Before proceeding,
ensure you have reviewed the [configuration guide](../how-to/configure.md) and
understood all available options.

Installation is straightforward: clone the repository, install dependencies, and
run the initialization command. The tool will detect your project type automatically
and generate a default configuration file tailored to your documentation engine.

Verify the installation by running a quick check on your docs directory. A passing
result confirms the setup is complete and your documentation is ready for review.
