# Blue-Team Demo

This fixture demonstrates clean documentation that passes all Zenzic checks and earns
the Obsidian Seal. Every link is relative, every section has substantial content, each
directory contains an index file, and no credentials appear anywhere in the source.

Run `zenzic check all` on this directory to see exit code 0 with the Obsidian Seal.
The contrast with the red-team fixture is the lesson: a clean exit means nothing without
the memory of the breach.
