# Welcome

This is the home page of the **Zensical Basic Example** — a minimal project
used to demonstrate how to use Zenzic with the [Zensical][zensical] build
engine.

[zensical]: https://zensical.org

## What this example shows

- A `zensical.toml` using the **`[project]` scope** (Zensical v0.0.31+).
- A `zenzic.toml` with `engine = "zensical"` activating `ZensicalAdapter`.
- A small but complete navigation tree to exercise orphan detection.
- Clean internal links using relative paths.

Navigate to the [Guide](guide.md) or the [API Reference](api/index.md) to
explore the example.
