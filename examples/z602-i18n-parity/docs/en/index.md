# Quick Start

This is the English baseline for the Z602 I18N_PARITY gallery example.
The Italian mirror at `docs/it/index.md` exists and matches this structure.
The Italian mirror of `docs/en/guide.md` is **intentionally absent** → Z602.

## Installation

Install Zenzic via pip or uv:

```bash
pip install zenzic
```

## Usage

Run the full check suite:

```bash
zenzic check all
```

## Configuration

Create `.zenzic.toml` in the project root with `docs_dir = "docs"`.
