# Avvio rapido

Questa pagina è la traduzione IT della pagina principale.
Il file `docs/it/guide.md` è **intenzionalmente assente** — Zenzic emette Z602.

## Installazione

Installa Zenzic tramite pip o uv:

```bash
pip install zenzic
```

## Utilizzo

Esegui il controllo completo:

```bash
zenzic check all
```

## Configurazione

Crea `.zenzic.toml` nella root del progetto con `docs_dir = "docs"`.
