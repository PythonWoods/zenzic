# Guide — v1.0.0

This is the guide section of the v1.0.0 documentation snapshot.

The v1.0.0 release introduced the core adapter protocol and the Virtual Site Map
engine. All link validation, anchor checks, and Shield scans were available from
this version onward. No build pipeline was required.

Return to the [v1.0.0 Introduction](../intro.md) or read the
[current Introduction](@site/docs/intro.mdx) for the latest version.
