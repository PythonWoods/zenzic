# Guide

This section covers installation, configuration, and deployment.

Zenzic validates all links in this guide tree without executing the Docusaurus
build pipeline. It reads `docusaurus.config.ts` as plain text to extract
`baseUrl` and `routeBasePath`, then maps each `.mdx` source file to its
canonical URL using pure Python.

Navigate to [Deployment](deploy.mdx) or return to the [Introduction](../intro.mdx).
